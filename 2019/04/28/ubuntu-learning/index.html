2 安装系统

  1. 要求主板支持 uefi, 并关闭主板上的安全启动

  2. 将下载的 lubuntu-18.04.2-desktop-amd64.iso 或其他版本镜像解压到 U 盘根目录，进主板 BIOS 并设置从 U 盘启动

  3. 安装时不要联网下载软件以防下载过慢

  4. 首选项->软件和更新->Ubuntu 软件->下载自 切换软件源到清华源

  5. 执行以下命令更新软件，选项一直选 Yes
