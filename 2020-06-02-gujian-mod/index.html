船新的游戏体验

素材提取

游戏解包软件 RPGViewer

用RPGViwer解包游戏数据，注意不同 zpk 文件要解包在不同的文件夹里，古剑一的解包数据软件不会自动分类，当然同样的文件名只是有些文件名 zpk 有数字的这种可以解包在同样的文件夹里。然后修改好文件后，你解包的哪个文件，就在游戏目录下该文件的同级目录下新建一个同名文件夹，然后把修改后的文件，按照解包的该文件所在的路径放到那个同名文件里的相同路径下。

古剑 MOD 辅助工具

将文件解压，得到两个文件GameServices.dll、GJPatch.ini和文件夹Patches，放到游戏Bin目录下。

例如：解包的是游戏目录Data\Objects文件夹里的characters.zpk文件，修改的是解包目录里的texture文件夹里的101-01.dds，那么你把修改后的101-01.dds放到游戏目录Data\Objects\characters\texture里。
然后开始游戏换上对应外装即可。

NVidia PS 插件

64 位

32 位

贴图修改完成后保存时注意勾选alpha 通道，存储格式选DXT3或DXT5。其中，DXT3 可很好地用于 Alpha 通道锐利、对比强烈的半透和镂空材质，适用于铁栏杆等 Alpha 边缘清晰的贴图。而 DXT5 则更适合 Alpha 通道柔和的材质，适用于头发等 Alpha 过渡柔和的贴图。

Alpha 通道的纯黑部分在游戏里不显示，纯白部分显示，灰色则为半透明。因此可以通过涂黑某些部分使其达到透明或者半透明效果。

模型预览工具

NifSkope

打开外装对应 nif 文件，在预览窗口选择组件，右键 Block -> Remove 查看效果，再修改贴图。
