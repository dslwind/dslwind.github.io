Firefox 新标签页弹出窗口

在地址栏输入about:config，将browser.link.open_newwindow.restriction更改为0，并确保将browser.link.open_newwindow设置为3。

另外，在较旧版本的 Firefox 里，可以通过选项-高级-更新，选择不检查更新来关闭自动更新；或者在about:config页面，将app.update.auto设置为false来关闭更新；新版本已移除上述两个选项。

官网下载

https://ftp.mozilla.org/pub/firefox/releases/

优先选择 ESR 版本，其中，Firefox 52.9.0esr 是最后一个支持 Windows XP 和 Windows Vista 的版本，Firefox 57 起使用 Quantum 引擎，在访问某些为旧浏览器设计的网站时，可以使用 Firefox 52.9.0esr版。
