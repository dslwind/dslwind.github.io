Firefox使用小技巧

发表于 2021-04-25 更新于 2021-04-27

每天一个分手搞机小技巧

Firefox 新标签页弹出窗口

在地址栏输入about:config，将browser.link.open_newwindow.restriction更改为0，并确保将browser.link.open_newwindow设置为3。

禁用自动更新

在 firefox 可执行文件所在目录中创建一个名叫distribution的文件夹，在distribution文件夹中新建policies.json的空白文档，使用文本编辑器打开policies.json，在其中输入下方内容并保存：

更多关于policy的介绍，参见 Firefox 的GitHub 页面。

另外，在较旧版本的 Firefox 里，可以通过选项-高级-更新，选择不检查更新关闭自动更新；或者在about:config页面，将app.update.auto设置为false；新版本已移除上述两个选项。

官网下载

https://ftp.mozilla.org/pub/firefox/releases/

优先选择 ESR 版本，其中，Firefox 52.9.0esr 是最后一个支持 Windows XP 和 Windows Vista 的版本，Firefox 57 起使用 Quantum 引擎，在访问某些为旧浏览器设计的网站时，可以使用Firefox 52.9.0esr版。
